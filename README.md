## 📖 使用指南喵

### 1. 试试预设的方法喵
- 页面已经分好类啦，找找你想打开的设置在哪一栏喵。
- 看到感兴趣的卡片，就大胆点一下 **“戳我试试喵~”** 按钮，看看能不能跳过去喵。
- 需要代码？点 **“复制一下喵”** 按钮，Intent 路径就到剪贴板了喵。

### 2. 添加你自己的独门方法喵
1.  翻到页面下面的 **“添加自定义Intent喵”** 表单。
2.  填上目标的 **包名** (比如 `com.android.settings`) 和 **完整类名**。
3.  给它起个**好记的名字**和**描述** (可选喵)。
4.  点击 **“添加喵~”**，它就会出现在 **“自定义喵”** 的分类里啦！随时可以测试、复制或删除它喵。

### 3. 换个心情喵 (主题与背景)
- 点击页面右下角的 **🌙/🌸** 按钮，可以在冷静的蓝色和可爱的粉色之间切换喵。
- 点击旁边的 **🎲** 按钮，可以随机换一张漂亮的背景图喵，每次都有新感觉！
